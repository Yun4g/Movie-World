
# 🎬 Movie World - Movie Browser App

Movie World is a modern web application built with **Next.js** that allows users to search, browse, and discover movies using the TMDB (The Movie Database) API and YouTube API. It features real-time search, detailed movie pages, trailers, and more.

## 🌟 Features

- 🔍 Search for movies by title
- 🎥 Watch movie  via YouTube

- ⚡ Fast and SEO-friendly with Next.js
- 🔐 Secure API integration with environment variables

## 🔧 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: Tailwind CSS
- **APIs**: TMDB API, YouTube API
- **Authentication**: JWT
- **Database**: MongoDB
